### Summary of Order: From smallest use/growth to largest

| type        | O(n)          |
|-------------|---------------|
| Constant    | 1             |
| Logarithmic | log(n)        |
| Linear      | n             |
| Log-Linear  | n log(n)      |
| Polynomial  | n<sup>a</sup> |
| Exponential | a<sup>n</sup> |
| Factorial   | n!            |

* a are integers or other real numbers > 0.
